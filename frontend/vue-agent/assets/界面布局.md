# 聊天界面布局规范

## 整体布局

```
┌─────────────────────────────────────┐
│           页面背景: 白色             │
│  #ffffff                            │
├─────────────────────────────────────┤
│                                     │
│         历史记录区域                 │
│         (可上下滚动)                 │
│         下对齐显示                   │
│         flex-direction: column      │
│         justify-content: flex-end   │
│                                     │
│                                     │
├─────────────────────────────────────┤
│           [输入框]                  │
│         水平居中                     │
│         底部固定                     │
└─────────────────────────────────────┘
```

## 结构层级

```
App.vue
├── 容器 (满屏 flex 布局, 白色背景)
│   ├── 历史记录区域 (flex: 1, overflow-y: auto)
│   │   └── 消息列表 (display: flex, flex-direction: column, justify-content: flex-end)
│   │       ├── 用户消息 (右对齐)
│   │       └── Agent消息 (左对齐)
│   └── 输入区域 (固定底部, 水平居中)
│       └── Input + 发送按钮
```

## CSS 核心样式

```css
/* App.vue */
.app-container {
  width: 100vw;
  height: 100vh;
  background-color: #ffffff;
  display: flex;
  flex-direction: column;
  overflow: hidden;
}

/* 历史记录区域 */
.chat-history {
  flex: 1;
  min-height: 0; /* 关键：让 flex 子元素的 overflow 能正常工作 */
  overflow-y: auto;
  display: flex;
  flex-direction: column;
  justify-content: flex-end;  /* 下对齐 */
  padding: 16px;
  padding-bottom: 100px;  /* 底部留白，避免被输入框遮挡 */
}

/* 消息列表 */
.message-list {
  display: flex;
  flex-direction: column;
  justify-content: flex-end;
  gap: 12px;
}

/* 输入区域 */
.input-area {
  position: fixed;
  bottom: 0;
  left: 0;
  right: 0;
  display: flex;
  justify-content: center;
  padding: 16px;
  background-color: #ffffff;
}
```

## 响应式要求

- 移动端：输入框宽度 100%
- 桌面端：输入框最大宽度 600px
- 历史记录区域占满中间剩余空间
