# 调查员与神话 - 规格文档

## 项目概述
- **项目名称**: 调查员与神话 (Investigator & Mythos)
- **类型**: 回合制桌游模拟
- **核心功能**: 卡牌战斗、资源管理、地图探索、章节流程控制

## 核心机制

### 章节流程区域
- **场景卡**：通往胜利的路径，需要消耗线索推进
- **密谋卡**：通往失败的路径，每个神话阶段+1密谋标记，标记满了触发负面效果

### 回合阶段
1. **神话阶段** - 第一回合跳过
   - 所有密谋卡+1标记
   - 触发神话事件
2. **调查阶段** - 3行动点
3. **敌军阶段** - 敌人行动
4. **整备阶段** - 回合结束处理

### 调查员属性
| 属性 | 说明 |
|------|------|
| 力量 (STR) | 影响攻击力 |
| 速度 (SPD) | 影响躲避率 |
| 智慧 (INT) | 影响调查成功率 |
| 胆识 (WIL) | 影响理智抵抗 |
| 生命 (HP) | 归零死亡 |
| 理智 (SAN) | 归零疯狂 |

### 行动（消耗行动点）
| 行动 | 消耗 | 效果 |
|------|------|------|
| 调查 | 1点 | 获得线索 |
| 攻击 | 1点 | 对敌人造成伤害 |
| 移动 | 1点 | 地图移动 |
| 出牌 | 1-2点 | 使用手牌能力 |
| 使用能力 | 2点 | 特殊效果 |

### 地图区域
- 6个地点：酒馆、森林、古堡、城镇、沼泽、祭坛
- 玩家和敌人在地图上移动
- 敌人出现在地图特定位置

### 牌组系统
- 牌组：抽牌堆
- 手牌：最多7张
- 弃牌堆：使用后放入
